# Demo 视频

本目录保存比赛提交用的演示材料。

- Demo 视频：[`ai-novel-to-screenplay-demo.webm`](./ai-novel-to-screenplay-demo.webm)
- 字幕文件：[`demo-subtitles.srt`](./demo-subtitles.srt)
- 配音脚本：[`demo-script.json`](./demo-script.json)

视频内容覆盖：

1. 打开本地 Web 工具。
2. 载入 3 章示例小说。
3. 展示文本分析、YAML 转换和大纲视图。
4. 展示来源覆盖、修订任务、篇幅规划和导出能力。
5. 说明规则引擎可离线运行，可选 AI 增强通过环境变量开启。

视频已经内置中文 AI 配音和画面字幕，`.srt` 字幕文件用于单独核对或平台字幕上传。
